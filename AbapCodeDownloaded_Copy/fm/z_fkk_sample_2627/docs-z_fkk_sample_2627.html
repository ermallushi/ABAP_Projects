Description:

SHORT TEXT
 INV: Supplement Invoicing Order for Source Documents of Category ACTIT

FUNCTIONALITY
 Event 2627 is processed when an invoicing order is created for an open
 item that is to be activated in invoicing (items with
 <DS:DE.AUGRS_KK>clearing restriction 8</>). In this event, you have the
 option of changing or suppressing this invoicing order.
 Invoicing order <ZH><ZK>C_INVTRIG</></> of <DS:DE.SRCDOCCAT_KK> source
 document category ACTIT</>, which was created based on the system
 configuration, is transferred to the interface. Except for the source
 document number, source document category, business partner, and
 contract account, you can change any other fields of this invoicing
 order.
 If you do not want the invoicing order to be created, initialize the
 parameter C_INVTRIG.
 
EXAMPLE
 
HINTS
 The event is called both at the posting of one of the documents to be
 activated in invoicing, and at the reversal of the invoice. If the event
 is called for an invoicing reversal, the invoicing document being
 reversed is made available in the I_REVERSED_INVDOC parameter.
