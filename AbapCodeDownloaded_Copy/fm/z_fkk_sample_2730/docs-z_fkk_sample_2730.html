SHORT TEXT
 INV: Ratenplan fällig stellen ( 'Auflösen in der Fakturierung')

FUNCTIONALITY
 Event 2730 is processed in invoicing when you execute the invoicing
 function <DS:DE.INVFUNC_INSTPLAN_DUE_KK>Set Installment Plan to Due</>.
 In this event, you can decide if an installment plan should be set to
 due within invoicing; in other words, you decide whether all of the
 installments of the installment plan that are not yet due at the time of
 invoicing should be due with the invoice.
 At the interface, the number of the installment plan is transferred in
 the parameter I_INSTPLAN_OPBEL. The not yet paid installments of this
 installment plan are transferred in C_INSTPLAN_FKKCL_TAB.
 The parameter C_SET_INSTPLAN_TO_DUE = 'X' specifies that the due dates
 of these installments are moved to the due date of the invoice. Set this
 parameter to C_SET_INSTPLAN_TO_DUE = ' ', if the installment plan is to
 be excluded from processing, that is, if it should keep its due dates.
