 Features
 The function module processed in event 2045 is called automatically
 during the posting of an interest document online (transaction FPI1) or
 during the interest run (FPINTM1). The module calls up one or more
 additional function modules before function modules for posting
 (FKK_CREATE_DOC or FKK_CREATE_DOC_AND_CLEAR) are called. All data is
 transferred for the interest document being posted.
 In this event you can update further data for the document, but you
 cannot report errors that the user could still correct.
