FUNCTIONALITY
 The function module processed in event 5070 is called up before posting
 during the writing-off process. Here, you can carry out special checks
 and changes to tax items.
 The write-off information and the items to be written off are provided.
 Table <ZH>T_FKKOPK</> contains the tax items for the write-off document.
 The additional parameters provide the information for the items written
 off and the information about whether there is a tax item in a document
 to be written off.
