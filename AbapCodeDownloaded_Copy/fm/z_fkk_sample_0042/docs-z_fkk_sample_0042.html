SHORT TEXT
 Beispiel 0042: Ausgleichsrestriktion bis zur Gesamtrechnung setzen

FUNCTIONALITY
 The function module called up for event 0042 decides whether clearing
 restriction '8' is set in an open item. If this clearing restriction is
 set, you cannot process (clear, charge interest, dunn) the item until
 the next regular bill is sent to the business partner.
 The item is printed for the first time on this bill.
 The data for the document header <ZH>I_FKKKO</> is available next to
 item <ZH>I_FKKOP</> as input.
 Value 'X' is returned to export parameter <ZH>E_AUGRS_INV</> if a
 clearing restriction is set. Otherwise the value SPACE is returned.
