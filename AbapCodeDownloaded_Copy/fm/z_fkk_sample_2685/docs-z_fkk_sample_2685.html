 INV: Correspondence - Create Invoicing

 trados = fi
 Features
 Event 2685 is triggered to create an invoicing print request.
 Dependencies
 You can print the invoicing document in <DS:FU.FKK_SAMPLE_2686>event
 2686</>. You have to maintain changes to the format of the
 correspondence requests in both events.
 Area of Implementation
 Correspondence
 Notes for Programming
 You can use the functions of function group FKK2 to create and read
 correspondence requests.
 For detailed information about implementing correspondence, see SAP note
 194899.
