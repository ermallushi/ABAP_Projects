Code listing for: Documentation - Z_COM_BPUS_BUPA_FOR_USER_GET

Description:

SHORT TEXT
 Determine User for Business Partner

FUNCTIONALITY
 You can use this function module to determine the business partner who
 is assigned to the user transferred via the central person.
 You can use the optional input parameters  IS_ALIAS and/or IS_USERNAME
 to assign the user. An existence check is performed for the user defined
 using the input parameter. If IS_ALIAS and IS_USERNAME are transferred,
 an additional check is performed to see whether the alias transferred is
 assigned to the user name transferred.
 If no user is assigned, the value available in the system variable
 SY-UNAME is read as user.
 If an error occurs, corresponding exceptions are triggered.
